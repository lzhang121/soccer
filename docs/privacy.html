プライバシーポリシー

サッカー速読（以下「本拡張機能」）

1. 概要

本拡張機能は、ユーザーが閲覧中のサッカーニュース記事を要約するツールです。 開発者はサーバーを運用せず、ユーザーの API キーで AI プロバイダーへ直接リクエストします。

2. 収集・送信するデータ

  • 記事本文・タイトル：要約のため、ユーザー端末から OpenAI / Claude / Groq / OpenRouter 等へ送信されます。
  • API キー：ユーザーが設定したキーを chrome.storage.local にのみ保存します。
  • 要約キャッシュ：同一 URL の要約結果を最大 24 時間、端末内に保存します。

3. 開発者が収集しないもの

  • 個人を特定する情報
  • 閲覧履歴の外部送信
  • 分析・広告トラッキング

4. 第三者サービス

ユーザーが選択した AI プロバイダーの利用規約・プライバシーポリシーが適用されます。 API 利用料は各プロバイダーからユーザーに請求されます。

5. データの削除

Options ページからキャッシュを削除できます。拡張機能をアンインストールすると、ローカル保存データも削除されます。

6. お問い合わせ

GitHub リポジトリの Issues よりご連絡ください。
